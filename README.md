# LOFO - Lost and Found Management System

LOFO (Lost It, List It, Find It) is a full-stack Lost and Found Management System built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js).

The platform allows users to report lost items, post found items, search available listings, upload item images, and connect with item owners. It provides a secure authentication system using JWT and stores data in MongoDB Atlas.

The goal of LOFO is to simplify the process of recovering lost belongings by creating a centralized digital platform where users can quickly report, search, and manage lost and found items.

## Features

- User Registration and Login
- JWT Authentication
- Create Lost Item Reports
- Create Found Item Reports
- Upload Item Images
- Search Items by Title
- Filter Items by Category and Status
- View Item Details
- Manage Personal Listings
- MongoDB Atlas Integration
- RESTful API Architecture
- Responsive User Interface
